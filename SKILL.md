# SKILL.md

This file defines the skills available to the AI agents.

## 1. Document Summarization
- Summarize long regulatory documents into concise overviews.
- Identify key claims and indications for use.

## 2. Risk Analysis
- Extract hazards, failure modes, and mitigations.
- Build risk matrices.

## 3. Testing Review
- Analyze bench, animal, and clinical testing data.
- Identify gaps in testing coverage.
